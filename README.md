# 🎰 Mini Stake - Python Casino Project

A feature-rich virtual casino platform built in Python.

## 🎮 Games
- 🎡 **Wheel Game** - Spin the wheel, win up to 5x with live animation
- 🎲 **Dice Game** - 4 modes including multi-dice round
- 🪙 **Coin Flip** - Heads or tails with double payout
- 🃏 **Blackjack** - Full blackjack with dealer AI, double down and split
- 💣 **Mines** - Reveal cells, grow multiplier, avoid bombs!

## ✨ Features
- 👤 Player login and registration
- 💾 Progress saved automatically
- 🏆 Leaderboard with top 10 players
- 🎨 Colorful terminal interface
- 📈 Win/loss tracking

## 🚀 How to Run
1. Make sure Python is installed
2. Install dependencies:
```
   pip install colorama
```
3. Clone the repository:
```
   git clone https://github.com/rushabhpandya/PythonProjectCasino
```
4. Run the game:
```
   python main.py
```
