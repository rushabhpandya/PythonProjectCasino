# 🎰 Mini Stake - Python Casino Project

A virtual casino platform built in Python with 5 fully playable games.

## 🎮 Games
- 🎡 **Wheel Game** - Spin the wheel and multiply your bet up to 5x
- 🎲 **Dice Game** - Guess the number, odd/even, or high/low
- 🪙 **Coin Flip** - Simple heads or tails with double payout
- 🃏 **Blackjack** - Full blackjack with dealer AI and double down
- 💣 **Mines** - Reveal cells to grow your multiplier, avoid the bombs!

## 🚀 How to Run
1. Make sure Python is installed
2. Clone the repository:
```
   git clone https://github.com/rushabhpandya/PythonProjectCasino
```
3. Run the game:
```
   python main.py
```
